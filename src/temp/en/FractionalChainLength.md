# Fractional Chain Length

$$FCL_{(x)} = ECL_{(x)} - ECL_{(z)}$$

- $z$ is the saturated fatty acid with the same number of carbons;

---

$$FCL_{(x)} = ECL_{(x)} - ECL_{(B)}$$

- $B$ is the saturated fatty acid with the same number of carbons;

## Abbreviations

- FCL

## Description

The fractional chain length is defined as the difference between the ECL value of the actual FAME molecule and the ECL value of the unbranched saturated molecule with the same number of carbons.
